### 三、功能描述
系统分为（身份）：普通用户、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 竞赛信息
5. 竞赛资讯
6. 个人中心

**教师功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 题目类型管理
5. 竞赛题库管理
6. 竞赛类型管理
7. 竞赛信息管理
8. 参赛申请管理
9. 竞赛评分管理
10. 参赛名单管理
11. 晋级名单管理
12. 获奖名单管理
13. 竞赛总结管理
14. 报销清单管理
15. 成绩申诉管理
16. 参赛信息管理
17. 往年成绩管理
18. 获奖情况管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 教师管理
4. 学生管理
5. 领队教师管理
6. 竞赛类型管理
7. 竞赛信息管理
8. 学院管理
9. 专业管理
10. 获奖情况管理（轮播图管理、竞赛资讯）
